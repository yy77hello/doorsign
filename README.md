

# Digital Door Sign System Enabling Two-Way Communication Between an Office Owner and Visitors

## Setup and Usage

- **Designed for Raspberry Pi or Linux systems**: 
  Add `flask_startup.sh` to system startup to automatically launch the webserver in kiosk mode on device boot.

- **Customizable QR Code**:
  Add your own email QR code by placing a `qrcode.png` file in the `/static/` directory.
